# 期末救命箱

面向大学生的课程资料与 DDL 智能整理助手。

「期末救命箱」不是网盘，而是一个课程资料作战台：把群聊通知、课程文件、作业要求、复习资料整理成按课程归档的资料库和待办清单。当前版本是一个前端可运行 MVP，重点跑通“上传/粘贴 -> 识别确认 -> 归档 -> 搜索”的完整闭环。

## 当前功能

- 首页 Dashboard：课程、资料、待办统计，近期 DDL，最近资料，课程概览
- 课程管理：课程列表、新建课程、课程详情页
- 添加资料：通过文件选择器读取文件名，只保存文件名和元数据，不上传真实文件
- 粘贴通知：输入群通知文本，使用本地 mock 规则识别结构化信息
- 识别确认 2.0：
  - 识别课程、事项标题、截止时间、提交方式、标签
  - 识别到未知课程时提示一键创建并使用
  - 显示截止时间识别依据
  - 对未识别字段给出提示，允许用户手动修正
- 待办追踪：确认通知后生成待办，支持完成 / 恢复待完成
- 全局搜索：首页搜索课程名、文件名、标签、通知原文、待办标题
- 本地持久化：使用 localStorage 保存课程、资料、通知和待办
- 演示数据：首次打开自动注入示例课程和资料

## 产品流程

1. 创建或选择课程，例如「数据库系统」「高等数学」「大学英语」。
2. 添加课程资料，保存文件名、课程、资料类型、标签和备注。
3. 粘贴群通知，例如作业要求、考试安排、展示通知。
4. 系统进行本地 mock 识别，提取课程、事项、DDL、提交方式和标签。
5. 用户在确认页检查和修改识别结果。
6. 确认归档后，系统创建通知记录和待办事项。
7. 用户在课程详情页查看资料、通知和待办。
8. 用户在首页用关键词搜索课程、资料、标签、通知和 DDL。

## 技术栈

- Next.js App Router
- TypeScript
- Tailwind CSS
- React
- lucide-react
- localStorage
- date-fns
- ESLint

当前版本不包含后端、登录注册、真实文件上传、真实 AI API、数据库、云存储、OCR、QQ/微信自动导入或学习平台接口。

## 本地运行

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

访问：

```txt
http://localhost:3000
```

如果 `localhost` 无法访问，可以尝试：

```txt
http://127.0.0.1:3000
```

## 验证命令

```bash
npm run lint
npm run build
node scripts/test-recognition.mjs
```

`scripts/test-recognition.mjs` 用来验证通知识别里的关键日期规则，例如：

- `下周三` 不会被误识别成 `周三`
- `下周1` 可以识别为下周一

## 数据说明

MVP 使用 localStorage 保存数据，主要 key：

```txt
final-rescue.appData.v1
final-rescue.recognitionDraft.v1
```

保存的数据包括：

- `courses`：课程
- `materials`：资料元数据
- `todos`：DDL 待办
- `notices`：通知记录

资料上传目前只读取文件名，不保存文件内容，也不会上传到服务器。

## 项目结构

```txt
app/          Next.js 页面路由
components/   UI 和业务组件
hooks/        数据 hook
lib/          类型、存储、识别、搜索、日期工具
scripts/      本地验证脚本
```
